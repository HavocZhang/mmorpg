技术架构

客户端 (TCP) → Nginx SLB (:7890, L4 least_conn) → 网关集群 (3 nodes, TCP :788x) ├─ 握手鉴权 (AES-256-GCM + Token) ├─ ReadLoop (解码 → 限流 → 路由) │ └─ gRPC UpstreamRouter → Logic Server (:50051) │ └─ ForwardMessage / PlayerOnline / PlayerOffline ├─ WriteLoop (16ms 合包窗口 → 三级优先级队列) │ └─ DownstreamDispatcher (精准下发 / 广播) ├─ Cluster (Redis PubSub 跨网关消息) │ ├─ NodeRegister (gate:nodes SET + gate:node:N TTL=30s) │ ├─ NodeHeartbeat (gate:heartbeat:N TTL=10s) │ ├─ CrossGatePubSub (channel: gate:cross) │ └─ RouteIndex (gate:route:UID → node_id, TTL=1h) ├─ Security (IP 限流 + 玩家限流 + 全局限流) └─ Admin (actix-web :909x, /health /stats /merge_stats /metrics) 可观测性 ├─ Prometheus (15s scrape, 12+ 指标) ├─ Grafana (15 面板, 4 行布局) └─ 告警 (8 规则: 连接/吞吐/限流/合包/内存/Session/gRPC/延迟)
